O HTML e suas tags

As tags de h1 até h6 são utilizadas para atribuir cabeçalhos as páginas html.

A tag p é utilizada para a criação de parágrafos na página.

A tag mark é utilizada realizar marcação em um texto.

A tag small reduz o texto que o envolve.

A tag i adiciona o estilo itálico ao texto.

A tag u adiciona um sumblinhado ao texto.

A tag strong adiciona um destaque ao texto.

A tag ol cria uma lista ordenada.

A tag ul cria uma lista não ordenada.

A tag li adiciona itens as lista.

A tag a adiciona um link aos elementos html.

A tag hr cria uma linha horizontal na página.

A tag sub cria uma miniatura do elemento abaixo.

A tag sup cria uma miniatura do elemento acima.

A tag blockquote cria um bloco de citação na página.
